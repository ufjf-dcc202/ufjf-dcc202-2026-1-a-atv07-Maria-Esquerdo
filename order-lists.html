Listas não Ordenadas

O que são: São listas numeradas (1, 2, 3... ou a, b, c...) que estabelecem uma sequência lógica ou cronológica entre os itens.

Quando usar: Quando a ordem dos elementos faz diferença, como em instruções passo a passo, rankings ou processos.

Exemplos:

Passo a passo:

  1. Ligue o computador.
  2. Abra o navegador.
  3. Acesse o site desejado.
  4. Faça login com seu nome de usuário e senha.
