Listas de Definição

O que são: São estruturas que associam cada termo a uma descrição ou definição, geralmente formatadas com o termo em destaque e a explicação logo abaixo ou ao lado.

Quando usar: Quando se deseja explicar conceitos, termos técnicos, glossários, ou perguntas e respostas.

Exemplos:

Glossário de termos técnicos na informática:

HTML
Linguagem de marcação utilizada para estruturar o conteúdo de páginas web.
CSS
Linguagem de estilos responsável pela aparência visual de páginas web.
JavaScript
Linguagem de programação que adiciona interatividade e dinamismo aos sites.
React
Biblioteca JavaScript para construção de interfaces de usuário baseadas em componentes.

Perguntas e respostas (FAQ):

O que são variáveis?
São espaços na memória do computador usados para armazenar dados temporariamente.
O que são funções?
São blocos de código reutilizáveis que executam uma tarefa específica.
O que são arrays?
São estruturas de dados que armazenam múltiplos valores em uma única variável.
O que são laços de repetição?
São estruturas que repetem um bloco de código enquanto uma condição for verdadeira.
